# Assets Directory

This directory should contain the following image assets:

## Required Assets

### icon.png

- Size: 1024x1024 pixels
- Format: PNG
- Purpose: App icon for iOS

### splash.png

- Size: Recommended 1284x2778 pixels
- Format: PNG
- Purpose: Splash screen shown during app launch

### adaptive-icon.png

- Size: 512x512 pixels
- Format: PNG with transparent background
- Purpose: Android adaptive icon (foreground)

### favicon.png

- Size: 48x48 pixels
- Format: PNG
- Purpose: Web favicon

## Note

For now, you can create simple placeholder images or use Expo's default assets.
To generate default assets, you can use online tools or create simple colored squares with the app logo/name.

For the powerlifting theme, consider using:

- Primary color: Orange/Red (#FF5722)
- Background: White/Dark
- Icon: Barbell or weight plate imagery
